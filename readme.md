# npmdash

fetch and display npm packages info, like downloads, stars, issues etc...

[Try it out](http://npm.devresources.site/)

### cli usage

```
npm i -g npmdash
npmdash -u <npm username>
```

### Tech Stack
- nodejs
- typescript
- puppeteer (for packages list)
- commander (cli)
- docker
- aws fargate
